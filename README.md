## Техническое задание.

## Установка:
## Выполнить команду:
- composer install .
- Переименовать файл .env.example в файл .env и настроить подключение к базе данных и пр.
- php artisan migrate --seed.
## Стартовая страница.
- public\index.php.

## Использованы :
- php 8
- Laravel 8
- MySQL 8
- composer
